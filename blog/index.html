Untitled (Rain)

I helped develop Python software for this art project with Owen Herbert and Jon D'alton, converting rain sculptures into musical pieces.

We used the astronomical crater detection algorithm pycda and midi data library miditime to map rain impact craters on physical scuptures to notes in a piece of music.
